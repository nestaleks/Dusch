
This repository contains a small responsive website built to resemble the structure of the referenced product page, styled with beige and light-brown tones. It includes:

- Header with logo and Impressum link
- Full-width hero banners
- Product/benefit sections
- Reviews section with an automatic slider
- Footer with logo and social links
- Mobile-first, responsive layout

How to use
- Open index.html in a browser.
- Impressum is available at impressum.html.

Development notes
- Styling lives in assets/css/style.css
- JavaScript (reviews slider) in assets/js/main.js
- Images/SVG in assets/img/

Plan
- The detailed plan and to-do log are stored under plans/ with the current date.

Premium theme (2025-11-25)
